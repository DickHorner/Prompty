# Icon Placeholders

The icons in this directory are SVG placeholders. For production, replace with proper PNG icons:

- icon-16.png (16x16)
- icon-48.png (48x48)  
- icon-128.png (128x128)

You can convert the SVG to PNG using tools like:
- ImageMagick: `convert icon-128.svg -resize 128x128 icon-128.png`
- Online converters
- Design tools (Figma, Illustrator, etc.)
